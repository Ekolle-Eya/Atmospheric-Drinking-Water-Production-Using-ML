# Atmospheric-Drinking-Water-Production-Using-ML
This is a predictive machine learning model use to determine drinking water production from Atmospheric Water Generator. The model uses two features (Temperature and Relative Humidity) to pridict water production.
The features in the data include Temperature, Humidity, Water Production_Day, Avg_Number of People and Avg_Number of Household.
